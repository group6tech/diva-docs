Adding a DIVA User Account

When adding a DIVA user account, you can grant the user access to one or both of the following:

  • DIVA Client
  • DIVA Fusion

Before You Begin

Before adding a user account, ensure that:

  • DIVA Fusion is running and you are signed in
  • you have the user's first name and last name
  • the user has chosen or been assigned a user name for the user account, and
  • (optional) you have the user's email address.

To add a DIVA user account:

  1. From DIVA Fusion's Users menu, click 'Users'.
    • Result: A list of users is displayed.
  2. Click New User.
    • Result: The User Information and Member Information fields are displayed.
  3. Complete all relevant user Information and Member Information fields and options.
    • Note: The fields marked with a red asterisk are mandatory.
    • Note: With the Create Member option selected (ticked), the user will be able to sign into DIVA Fusion.
    • Note: With the DIVA User option selected (ticked), the user will be able to sign into DIVA Client and access the [?drives?]. You can restrict access to drives later on.
  4. Click Save User
    • Result: Assuming the user account does conflict with another user account, the user account is created.
  5. Select (tick) the 'Is Approved' option. How about "Enable User Account"
    • Note: If selected, the user account will be enabled and will be ready to use. If deselected, the user account will be disabled and the user will not be able to use it to sign into a DIVA product.
  6. Click Save User. [?How about 'Click Save." ??]
    • Result: The changes you made are saved, and the user account is enabled.

Modifying a User's Access to One or More Drives

You can:

  • deny the user access to one or more drives
  • Note: When new user accounts are set up, access to drives is automatically denied.
  • grant read-only access to one or more drives
  • Note: The user can view, and copy files to another drive, but cannot save this files to read-only drive.
  • grant read/write access to one or more drives
  • Note: The user has full access to the drive and can save this files to it.

Granting a User Permission to Save Files (Write) to a Drive

Before You Begin

Before modifying a user's access to one or more drives, ensure that:

  • DIVA Fusion is running and you are signed in
  • an account for the user has been added, and
  • Note: To do this, see "Adding a DIVA User Account"
  • the drives that you are going to grant the user access to have been set up.
  • Note: If you need to set up the drives, for assistance, see Drives.

To modify a user's access to one or more drives:

  1. From DIVA Fusion's Users menu, click 'Users'.
    • Result: A list of users is displayed.
  2. From the list, click the relevant user account.
    • Result: The user account's properties are displayed.
  3. Click Drives.
    • Result: A list of drives is displayed.
  4. For the drives in question, select the relevant options.
    • Note: Options include Denied, Read and Write.
    • Note: You cannot mount a drive for the user from this page. If there is a tick, the user has mounted the drive. If there is a cross, the drive is not mounted for that user. [?Is this correct? ?]
  5. Click Save Permissions.
    • Result: The changes you made are saved.

What's Next?

Remember to notify the user of the changes.

Forcing a User to Sign Out of DIVA Client

If, for example, a user hasn't signed out, you can force the user's user account to sign out of DIVA client.

NOTICE! Forcing a User Account to Sign Out of DIVA Client
Only force a user to sign out of DIVA Client as a last resort. [?Are there any potential negative consequences like lose of data??]

Before you begin

Before forcing a user to sign out of DIVA client:

  • where practicable, let the user know that they may need to save their work [?Is this correct? ?]

To force a user to sign out of DIVA Client:

  1. From DIVA Fusion's Users menu, click 'Users'.
    • Result: A list of users is displayed.
  2. From the list of user accounts, click the relevant checkboxes.
    • Result: The users' accounts are selected.

Deleting a User Account

Only delete a user account if you are certain it will no longer be needed. If it's possible that the user will require the account again, you can disable the user account for now.

NOTICE! Deleting a User Account
Before deleting the user account, ensure all of the user's files that you want to keep have been [?transferred?] to another user account. Deleting a user account automatically deletes the media files and project files in the user's folders.

Before You Begin

Before deleting a user account, ensure DIVA Fusion is running and you are signed in.

To delete a user account:

  1. From DIVA Fusion's Users menu, click 'Users'.
    • Result: A list of users is displayed.
  2. From the list, click the relevant user account.
    • Result: The user account's properties are displayed.
  3. Click Delete User.
    • Result: You will be asked to confirm that you want to delete the user account.
  4. Click Delete User.
    • Result: The user account is deleted.

Resetting a User's Password

Before You Begin

Before resetting a user's password, ensure DIVA Fusion is running and you are signed in.

To reset a user's password:

  1. From DIVA Fusion's Users menu, click 'Users'.
    • Result: A list of users is displayed.
  2. From the list, click the relevant user account.
    • Result: The user account's properties are displayed.
  3. Click Membership.
  4. Click Change Password
    • Result: Two password fields are displayed.
  5. In the New Password and Confirm Password fields, type the new password.
  6. Press Change Password.
    • Result: The password is changed.

What's Next?

Remember to notify the user of the new password.
