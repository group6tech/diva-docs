About Favourites

Favourites provide DIVA Client users with easy access to the drives they use most often. A favourite is a customised collection of DIVA drives that can be mounted in one go by the DIVA Client user. This saves users time because by using favourites, users don't have to locate and mount the drives they require one-by-one.

A favourite can be set up for:

  • all DIVA Client users
  • a particular group of users, e.g., a project team, or
  • multiple groups of users.

Displaying & Editing a Favourite

When displaying a favourite you can:

  • change the name of the favourite
  • edit the favourite's description
  • enable or disable the favourite
  • change how the favourite is shared
  • Note: You can share the favourite with all DIVA CLient users, or restrict the favourite to members of one or more user groups.
  • add and remove links to drives, and
  • Note: That is, you can change which drives are to be mounted when the favourite is used.
  • map a drive to a preferred drive letter (MS Windows only)
  • The preferred letter will be used if it is available when the drive is mounted. If it is not available, another letter will be mapped to the drive.
  • if the favourite is not shared with all DIVA Client users, you can assign the favourite to different user groups.

Before You Begin

Before editing a favourite, ensure DIVA Fusion is running and you are signed in.

To display the favourite:

  1. On DIVA Fusion's menu, click Favourites.
    • Result: A list of favourites is displayed.
    • Note: If the list of favourites is long, use the search tools on the left to narrow down your search.
  2. Click the relevant favourite's name.
    • Result: The favourite's properties are displayed along with a Drive tab and possibly a Permissions tab.

Adding a Favourite for All DIVA Client Users

Setting up a favourite for all DIVA Client users to use is straight forward to do. Once set up, all DIVA Client Users will be able to use it to mount the drives that are assigned to it.

Before You Begin

Before adding a favourite for all DIVA Client users, ensure:

  • DIVA Fusion is running and you are signed in, and
  • the drive or drives that will be mounted by this favourite have been set up
  • Note: One or more drives needs to be set up (added to a volume) before they can be assigned to the favourite. For information on setting up a drive see DIVA Fusion > Drives

To add a favourite for all DIVA Client users:

  1. On DIVA Fusion's menu, click Favourites.
    • Result: A list of favourites is displayed.
  2. Click New Favourite.
    • Result: The new favourite's properties are displayed.
  3. Complete the Favourite Name field and if required, the Description field.
    • Note: Give the favourite a unique name, for example, a name that encapsulates the drive or drives that you will be giving everyone access to.
    • Note: If providing a description, provide enough information to distinguish this favourite from any others.
  4. Ensure the Enabled checkbox is selected (ticked).
    • Note: This checkbox should already be selected (ticked).
  5. Ensure the Shared checkbox is selected (ticked).
    • Note: This checkbox should already be selected (ticked).
  6. Click Save Favourite.
    • Result: The Drives tab is displayed next to the Properties tab.
  7. Click Drives.
    • Result: A drop down menu of drives is displayed.
  8. From the drop down menu, select a drive, then click Add Drive.
    • Result: The selected drive is added to a list of drives.
  9. (Optional) Assign a preferred drive letter to the drive.
    • Note: To do this, from the drop down menu, select a letter. If a letter has currently being used by a mounted drive, this letter will not be available for selection.
    • Note: This step is only relevant to MS Window's users. If you are using a Mac, there is no need to assign a letter to a drive.
    • Note: If you assign a preferred letter to the drive, then when users mount the drive, the preferred letter will be mapped to the drive. If the drive letter you selected for this drive is already mapped to a mounted drive, another letter will automatically be mapped to this drive.
  10. Do you want to add another drive to the favourite?
    • If Yes, go to Step 8.
    • If No, go to 'What's Next?' (below).

What's Next?

On DIVA Fusion's menu, click Favourites to see your new favourite listed along with any others. Any DIVA Client user who has been permitted access to the favourite's drives will be able to use the favourite to mount these drives.

Adding a Favourite for One or More Groups of DIVA Client Users

Sometimes you will want to set up a favourite for a small group of users, for example a project team.

Before You Begin

Before adding a favourite for one or more groups of DIVA Client users, ensure:

  • DIVA Fusion is running and you are signed in
  • the drive or drives that will be mounted by this favourite have been set up, and
  • Note: One or more drives needs to be set up (added to a volume) before they can be assigned to the favourite. For information on setting up a drive see DIVA Fusion > Drives
  • you have set up the group of users for the favourite you are about to add
  • Note: The group needs to be set up before a favourite can be assigned to it. For information on setting up a group see DIVA Fusion > User Accounts > Setting Up a User Group

To add a favourite for one or more groups of DIVA Client users:

  1. On DIVA Fusion's menu, click Favourites.
    • Result: A list of favourites is displayed.
  2. Click New Favourite.
    • Result: The new favourite's properties are displayed.
  3. Complete the Favourite Name field and if required, the Description field.
    • Note: Give the favourite a unique name, for example, the name of a project that the group will be working on.
    • Note: If providing a description, provide enough information to distinguish this favourite from any others.
  4. Ensure the Enabled checkbox is selected (ticked).
    • Note: This checkbox should already be ticked.
  5. Ensure the Shared checkbox is not selected (not ticked).
    • Note: This checkbox is automatically selected, so you will need to deselect it.
  6. Click Save Favourite.
    • Result: The Drives tab and Permissions tab is displayed next to the Properties tab.
  7. Click Drives.
    • Result: A drop down menu of drives is displayed.
  8. From the drop down menu, select a drive, then click Add Drive.
    • Result: The selected drive is added to a list of drives.
  9. (Optional) Assign a preferred drive letter to the drive.
    • Note: To do this, from the drop down menu, select a letter. If a letter has currently being used by a mounted drive, this letter will not be available for selection.
    • Note: This step is only relevant to MS Window's users. If you are using a Mac, there is no need to assign a letter to a drive.
    • Note: If you assign a preferred letter to the drive, then when users mount the drive, the preferred letter will be mapped to the drive. If the drive letter you selected for this drive is already mapped to a mounted drive, another letter will automatically be mapped to this drive.
  10. Do you want to add another drive to the favourite?
    • If Yes, go to Step 8.
    • If No, go to Step 11.
  11. Click Permissions.
    • Result: A list of user groups is displayed.
    • If there is no list of groups, you can come back later to assign the favourite to the group after you have set the group up.
  12. From the drop down menu, select a user group, then click Add Group.
    • Result: The selected group is added to a list of groups that can use this favourite.
  13. Do you want to assign this favourite to another group.
    • If Yes, go to Step 12.
    • If No, go to Step 'What's Next?' below.

What's Next?

On DIVA Fusion's menu, click Favourites to see your new favourite listed along with any others. Any DIVA Client user who has been permitted access to the favourite's drives will be able to use the favourite to mount these drives.

Deleting a Favourite

When a favourite is no longer required, you can delete it. Deleting a favourite does not result in any loss of data.

Before You Begin

Before deleting a favourite, ensure:

  • DIVA Fusion is running and you are signed in, and
  • the favourite is no longer required.
  • Note: Once a favourite is deleted, the deletion cannot be undone. if you accidentally delete the wrong favourite, you will need to add a replacement favourite from scratch.

To delete a favourite:

  1. On DIVA Fusion's menu, click Favourites.
    • Result: A list of favourites is displayed.
    • Note: If the list of favourites is long, use the search tools on the left to narrow down your search.
  2. Click the relevant favourite's name.
    • Result: The favourite's properties are displayed.
  3. At the bottom of the page, click Delete Favourite.
    • Result: You will be asked to confirm the deletion.
  4. Click Delete Favourite.
    • Result: The favourite is deleted, and the list of favourites is displayed.
