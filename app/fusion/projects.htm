Projects

With DIVA Client's shared projects, a user can drag and drop bins within Avid's file manager to:

  • hide the bins from other users by placing them at root level in Avid's file manager
  • provide other users with view-only access of the user's files by placing the bins in the user's own folder
  • access other users folders to open files as read-only files, and
  • using the unprotected folder, transfer bins to other users.

Sharing a New Project

Sharing a user's project with other users requires you to perform a range of tasks across DIVA Fusion (or DIVA Client) and Avid.

Before You Begin

Before sharing a new project, ensure:

  • you are signed into DIVA Fusion
  • the relevant project drive is mounted in DIVA Client, and
  • you are signed into Avid and the Select Project window is displayed

To share a new project:

  1. In DIVA Fusion, enable the relevant project drive.
    • Note: This is the drive on which you want to share a new project. It's status must be 'Online'.
  2. On Avid's Select Project window, click External.
  3. Click the folder icon and locate the relevant DIVA project drive.
  4. Click New Project.
    • Result:The New Project window is displayed.
  5. Complete the new project's settings.
    • Note: You can provide the project with a meaningful name and adjust a range of settings, e.g., format, raster dimension and colour space.
  6. On Avid's New Project window, click OK,
    • Result: The Select Project window is displayed.
    • Note: Keep this window open for now.
    • Important: Do not open the new project in Avid just yet. Opening the project in Avid too soon may prevent the project from being shared.
  7. On the DIVA Fusion menu, click Projects.
    • Result: The new project is listed on the page.
    • Note: This will be listed as a project that is not shared. The project will have a blue Share button.


  8. Click the Project's Share button.


    • Result: You will be asked to confirm that you want to share the project


  9. Click Share.
    • Result: The project is shared with all the users of the drive. A message is displayed confirming that the new shared project is available on the relevant volume.


    • Note: Ensure this message is displayed before proceeding. If it does not display within a minute or two, check to see if the relevant job has been run on the server.
  10. On Avid's Select Project window, select the new project, then click OK.
    • Result: The project is opened in Avid Media Composer. The first bin and a range of folders is displayed in Avid's file manager.

What's Next?

The project is ready to use. You can use Avid's file manager to drag and drop bins from one folder to another.

Restoring Users' Access to a Shared Project

If users are having problems opening other users' bins, you can repair a project to fix the problems. Repairing a project will restore links and permissions.

Before You Begin

Before restoring users' access to a shared project, ensure DIVA Fusion is running and your are signed in.

To repair users' access to a shared project:

  1. On the DIVA Fusion menu, click Projects.
    • Result: A list of projects is displayed. Each project will have trash and wrench icons.


  2. Click the relevant shared project's wrench icon.


    • Result: You will be asked if you want to repair the project.
  3. Click Repair.
    • Result: Users' access to the project drive is restored.
    • Note: A job that repairs the project is created, queued and run in the background. Depending on the size of the project, this may take anything from a few seconds to 1 minute.

What's Next?

You can confirm that the repair took place by checking the list of jobs. To do this, on the Fusion menu, click Server, then click Jobs. The job's status when completed will be green.

Deleting a Shared Project

When deleting a shared project, the project files are deleted. The project's media is not deleted.

Before You Begin

Before deleting a shared project, ensure DIVA Fusion is running and your are signed in.

To delete a shared project:

  1. On the DIVA Fusion menu, click Projects.
    • Result: A list of projects is displayed. Each project will have trash and wrench icons.


  2. Click the relevant shared project's trash icon.


    • Result: You will be asked if you want to confirm the deletion.


  3. Click Delete.
    • Result: A job that deletes the shared project is created, queued and run in the background. On deletion of the shared project, a message will be displayed informing you that the shared project has been deleted.


    • Note: The shared project's media is not deleted; only the project's files are deleted.
