Sharing a New Project

Sharing a project successfully requires you to perform a range of tasks across DIVA Client and Avid.

Before You Begin

Before sharing a new project, ensure:

  • you are signed into DIVA Client
  • the relevant project drive is mounted in DIVA Client, and
  • you are signed into Avid and the Select Project window is displayed

To share a new project:

  1. In DIVA Client, mount the relevant project drive.
    • Note: This is the drive on which you want to share a new project.
  2. On Avid's Select Project window, click External.
  3. Click the folder icon and locate the relevant DIVA project drive.
    • Note: Find the drive that was mounted in DIVA Client.
  4. Click New Project.
    • Result:The New Project window is displayed.
  5. Complete the new project's settings.
    • Note: You can provide the project with a meaningful name and adjust a range of settings, e.g., format, raster dimension and colour space.
  6. On Avid's New Project window, click OK,
    • Result: The Select Project window is displayed.
    • Note: Keep this window open for now.
    • Important: Do not open the new project in Avid just yet. Opening the project in Avid prematurely may prevent the project from being shared.
  7. Display DIVA Client.
    • Result: A message is displayed at the bottom of the DIVA Client window informing you that DIVA has found the project.
    • Note: This message is displayed at the bottom of the DIVA window. Ensure this message is displayed before proceeding.
  8. On the DIVA Client menu, click Projects.
    • Result: The new project is listed on the page.
    • Note: This will be listed as a private project. The project will have a Share button.


  9. Click Share.
    • Result: The project is shared with all the users of the drive. The order in which projects are onscreen listed may change. A message is displayed at the bottom of the DIVA Client window confirming that the new shared project is available on the relevant volume.


    • Note: Ensure this message is displayed before proceeding. If it does not display within a minute or two, contact your DIVA Fusion administrator and ask if the relevant job has been run on the server.
  10. On Avid's Select Project window, select the new project, then click OK.
    • Result: The project is opened in Avid Media Composer. The first bin and a range of folders is displayed.

What's Next?

The project is ready to use. You can use Avid's file manager to drag and drop bins from one folder to another. For more information, see 'About Projects in Avid Media Composer' below.

About Project Folders in Avid Media Composer

Project Folders in Avid Composer's File Manager.

Prevent Other Users from Saving Changes to Your Files

To prevent other users from modifying files in your bins, use Avid's file manager to move the bins to your folder. Your folder is named after your DIVA Client username.

Note: Other users will be able to see and open the files in your folder, but will not be able to save any changes back to the bins in your folder.

Note: Likewise, if another user moves their bins to their own folder, you will be able to open them, but won't be able to save changes back to the bins in their folders.

Hiding Your Files from Other Users

You can hide your bins from other users by placing them at root level. At this level, your files will only be seen by you.

Note: You cannot use DIVA Client to archive/export files that are stored at root level.

Transferring Files between Users

Only use the Unprotected folder to swap files with other users. Ensure that files are not worked on while they are in the Unprotected folder. If someone places their files in the unprotected folder for you to work on, move the files from the unprotected folder and place them at root level or within your folder.

NOTICE! Working with Unprotected Files.
Do not work with bins while they are in the Unprotected folder. Avid is not multi-user aware. While bins are in the Unprotected folder, other users can overwrite any changes you make and vice versa. Overwritten work cannot be recovered.

Archiving an Avid Project (Export)

On MS Windows, projects can be exported to an archive folder for archiving at the end of a project, or for back-up purposes. You can archive users' folders and unprotected folders. Bins at the root level are not archived.

Archive / Export

Before You Begin

Before archiving an Avid project, ensure:

  • DIVA Client is running and your are signed in, and
  • the bins that you want to archive are not at root-level.
  • Note: Ensure all bins to be archived are stored within users folders or the project's Unprotected folder.

To archive an Avid project:

  1. On the DIVA Client menu, click Projects.
    • Result: A list of projects is displayed.
    • Note: This will be listed as a private project. The project will have a Share button.
  2. Click the relevant project's Export button.
    • Result: The Browse for Folder window is displayed.
  3. Locate the destination folder or make a new folder in a directory of your choice.
  4. Click OK
    • Result: A message is displayed informing you that the project was saved to the designated folder.
    • Note: The complete file path is specified in the message. The exported folder replicates the directory structure as seen in Avid, with copies of the project's files in copies of the users' folders. Any of the project's files that are at root level and in the Unprotected folder are also archived.
