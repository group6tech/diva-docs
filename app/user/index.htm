Overview

DIVA Client

DIVA is a media server designed and built for editors to streamline, store, share and simplify your Avid Media Composer environment.

Spend more time editing, less time setting up, and collaborate on projects with ease.

Signing In & Out of DIVA Client

To sign in to DIVA Client:

  1. Launch DIVA Client.
    • Note: To do this, on your desktop, double-click DIVA Client
  2. In the Username field, type your username.
    • Note: Your user name is not case-sensitive.
  3. In the Password field, type your password.
    • Note: Passwords are case sensitive. Type your password using the correct combination of upper and lowercase letters.
  4. Select or deselect Remember Me.
    • Note: If you select this option, you will automatically be signed into DIVA next time you launch DIVA. This assumes that you exit DIVA without logging out. If you log out, you will need to sign in. If you deselect this option, on launching DIVA, you will be required to sign in.
  5. Click Sign In.
    • Result: The Favourites page is displayed. Any favourites that you have access to will be displayed here.
    • Note: You can use a favourite to mount one or more drives in one go. The favourites will have been set up by your Fusion Administrator.

To sign out of DIVA Client:

Sign Out

  1. From the username drop down menu (top-right of window), click Sign Out.
    • Result: The Sign In fields are displayed.

Glossary of Basic Terms

The following list describes some of the terms that you may find useful when using DIVA Client.

For a comprehensive list of terms see Glossary of Advanced Terms

Drive

The drives created for use in DIVA are where you save and work with your files. If mounted on the computer that is being used and your user account has been granted full access to the drive, the drive is available to save and work on your files. If your user account has been granted read-only access to the drive, you can view files and copy files from the read-only drive to another drive.

The following picture shows a hard disk (storage device), like the one in your computer or server. On this volume, three drives have been set up; one of each type.

There are three types of drive.

  • Basic Drive
  • Avid Media Drive.
  • Avid Projects Drive

If you have full access to a Basic drive, you can add files of any type.

If you have full access to an Avid Media drive, you can only save and work with Avid MXF files on this drive.

If you have full access to an Avid Projects drive, you can only save and work with Avid Media Composer projects and bins.

Password

A password that is unique to each user account.

Note: You will require a password to sign in.

User
A user is a person who has been set up with a user account to use DIVA. Each user has a unique username. Each users ability to perform tasks and access drives is restricted by the user account.
Username

A code that is unique to each user of DIVA.

Note: You need your username and password when signing into DIVA. If you do not have a username or password, contact someone within your company who has administration rights to set you up with one.
