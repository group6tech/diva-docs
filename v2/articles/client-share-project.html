Share a project

Sharing a project successfully requires you to perform a range of tasks across DIVA Client and Avid.

Before you begin

Before sharing a new project, ensure:

  • you are signed into DIVA Client
  • the relevant project drive is mounted in DIVA Client, and
  • Avid Media Composer is open and the Select Project window is displayed

Share a new project:

  1. On Avid's Select Project window, click External.
  2. Click the folder icon and locate the relevant DIVA project drive.

    Note</strong>: Find the drive that was mounted in DIVA Client.

  3. Click New Project.
  4. Complete the new project's settings.

    Note: You can provide the project with a meaningful name and adjust a range of settings, e.g., format, raster dimension and colour space.

  5. On Avid's New Project window, click OK.

    Important: Do not open the new project in Avid just yet. Opening the project in Avid prematurely may prevent the project from being shared.

  6. Return to DIVA Client. A message is displayed at the bottom of the DIVA Client window informing you that DIVA has found the project.

    Note: This message is displayed at the bottom of the DIVA window. It may take a few moments for DIVA to discover the new project. Ensure this message is displayed before proceeding.

  7. On the DIVA Client menu, click Projects.
  8. Click Share for your new project.
  9. The project is shared with all the users of the drive. The order in which projects are onscreen listed may change. A message is displayed at the bottom of the DIVA Client window confirming that the new shared project is available on the relevant volume.
  10. On Avid's Select Project window, select the new project, then click OK.

Result: The project is opened in Avid Media Composer. The first bin and a range of folders is displayed.

What’s next?

The project is ready to use. You can use Avid’s file manager to drag and drop bins from one folder to another. For more information, see sharing bins in Avid Media Composer.
