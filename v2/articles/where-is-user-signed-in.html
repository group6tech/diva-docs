See where a user is signed in

You can easily view what edit suite a DIVA user is signed into.

  1. From DIVA Fusion's Users menu, click Users.
  2. Click on a user which is currently marked as being online.
  3. Click the Membership tab.
  4. See the name of the edit suite where they are signed in next to Signed into.
