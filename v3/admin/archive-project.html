Archive a project

Archiving a project creates a Zip file of all the Avid MediaComposer project files. See more about project archiving.

Archive a project

  1. On the DIVA Fusion menu, click Projects.
  2. Find the project to archive.

    Note: You can use the search and filter tools on the left to find a specific project.

  3. Click the button on the project's row.
  4. A confirmation is displayed. You have the option to Archive, Archive & Delete, or Cancel.

    Archive only creates a zip archive of the project files.

    Archive & Delete also deletes the project from the drive once the archive has been created.

    Cancel closes the dialog.

  5. Click one of the buttons to create the archive or close the dialog.

Result: A job is created to archive (and delete if chosen) the project. The zip file will soon be available on the drive on which the project resides.
