Manually backup a project

There may be times where you wish to manually backup a project. It could be that youâ€™re about to make major changes to it, and want a specific point in time to go back to, or you might not be running automatic backups at all.

Backup a single project

  1. On the DIVA Fusion menu, click Projects.

  2. Locate the project you wish to backup.

  3. Click the Tools button on the row of the project.

  4. Click Backup.

Result A job is created to backup the project.

Backup multiple projects

  1. On the DIVA Fusion menu, click Projects.

  2. Check the projects you wish to backup.

  3. At the bottom of the project list, click Backup.

  4. A confirmation dialog will appear, listing the projects you have selected. Click Backup.

Result A job is created to backup each selected project.
