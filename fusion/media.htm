Media

Avid Media files are stored in folders that are organised by user account. That is, on any Avid Media drive, media files can be spread across one or more users' folders. Sometimes, e.g., at the end of a project, you may want to assign the job of tidying up and archiving these files to one person. To achieve this, all the relevant media files that may currently be spread across other users' folders, can be moved under one user account.

Note: The ability to use DIVA Fusion to move media files is restricted to Avid Media drives. You cannot use DIVA Fusion to move files from Basic or Avid Project drives.

Moving Multiple Users' Media to One User Account

Before You Begin

Before moving multiple users' media files to one user account, ensure:

  • DIVA Fusion is running and you are signed in
  • there is at least one user account with write access to the media drive from which media will be moved, and
  • for the user account to which the media will be moved, that this user account has read or write access.
  • you have saved any permissions changes that you have made.

NOTICE! Moving Media Files
Only move a user's media files when you are certain the user has no more use for them. All the user's media files on the selected drive will be moved and this cannot be undone.

To move multiple users' media to one user account:

  1. From DIVA Fusion's Drives menu, click Drives.
    • Result: The DIVA Drives page is displayed.
  2. Locate and click the relevant media drive.
    • Note: If the list of drives is long, use the search tools on the left to narrow down your search.
    • Note: Ensure you have clicked a media drive; not a basic or projects drive.
    • Result: The drive's properties are displayed.
  3. Click Permissions.
    • Result: A list of user accounts and their permissions are displayed.
  4. Using the checkboxes, select the user accounts from which media files will be moved.
    • Note: In this example, I'm going to move the media files from Johnny's and Freddie's user accounts to Sam's user account.


  5. Click Move Media.
    • Result: A window lists the selected user accounts from which the media will be moved and a drop down menu from which the recipient user account can be selected.
  6. From the drop down menu, select the relevant user account.
    • Note: This is the user account to which the media files will be moved.


    • Note: If there are no user accounts with write access, the drop down menu and 'Move Media' button will not be displayed. In this case, you can either cancel the process or click 'Leave in Place', then grant the relevant user access, save your changes to the permissions and try again.
  7. Click Move Media
    • Result: The media on the drive is moved to a folder under the control of the relevant user account.

Moving One User's Media to Another User's Account

Before You Begin

Before moving a users' media files to another user's account, ensure:

  • DIVA Fusion is running and you are signed in
  • there is at least one user account with write access to the media drive from which media will be moved, and
  • for the user account to which the media will be moved, that this user account has read or write access.

NOTICE! Moving Media Files
Only move a user's media files when you are certain the user has no more use for them. All the user's media files on the selected drive will be moved and this cannot be undone.

To move a users' media files to another user's account:

  1. From DIVA Fusion's Drives menu, click Drives.
    • Result: The DIVA Drives page is displayed.
  2. Locate and click the relevant media drive.
    • Note: If the list of drives is long, use the search tools on the left to narrow down your search.
    • Note: Ensure you have clicked a media drive; not a basic or projects drive.
    • Result: The drive's properties are displayed.
  3. Click Permissions.
    • Result: A list of user accounts and their permissions are displayed.
  4. Click the relevant Move Media button.
    • Note: A Move Media button is displayed to the right of each user account line.
    • Note: In this example, I'm going to move the media files from Johnny's user account to Sam's user account.


    • Result: A window identifies the user account from which the media will be moved and a drop down menu from which the recipient user account can be selected.
  5. From the drop down menu, select the relevant user account.
    • Note: This is the user account to which the media files will be moved.


    • Note: If there are no user accounts with write access, the drop down menu and 'Move Media' button will not be displayed. In this case, you can either cancel the process or click 'Leave in Place', then grant the relevant user access, save your changes to the permissions, and try again.
  6. Click Move Media
    • Result: The media on the drive is moved to a folder under the control of the relevant user account.
