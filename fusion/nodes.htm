Nodes

A node consists of one or more volumes. An example of node is a computer or server that contains one or more volumes (storage devices). These storage devices may be internal (built into a server or computer), or external (connected to a server or computer).

An example of a node (server) and its volumes (storage devices)

Note: If one of the node's volumes, e.g., hard disk, is assigned to a pool, the volume is called a 'DIVA Volume'.

Creating a Node

Before You Begin

Before creating a node, ensure DIVA Fusion is displayed in your browser and you are signed in:

To create a node:

  1. From the Server menu, click Nodes.
    • Result: A list of nodes is displayed.
  2. Click 'New Server Node'.
    • Result: The New Node page displays a Hostname field and an IP Address field.
  3. In the Hostname field, type the name that you want to call the host.
    • Note: This is the actual host name of the computer or server on the network. Ensure there are no spaces in the host name.
  4. (Optional) In the IP Address field, type the IP address.
    • Note: You can leave this field empty. If you leave this field empty, an IP address will be detected on first connection.
  5. Ensure the Enabled option remains selected (ticked).
    • Result: The Enabled option will already be selected.
  6. Click 'Create Server Node'.
    • Result: The node is created and the browser displays the node's properties.
    • Note: Much of the information on this page is missing because the node is not yet connected.

Connecting a Node

Before You Begin

Before connecting a node, ensure:

  • the node has been created, and
  • To create the node, see Creating a Node (above).
  • DIVA Node is running
  • Diva Node, along with DIVA Server should already be running. The DIVA Node window and DIVA Server window are windows with green text on a black background. If DIVA Node is not running, on your desktop, double-click DIVA Node.
  • Note: The first time DIVA Node is run, the configuration window is displayed.

To connect a node:

  1. If the DIVA Node Configuration window is not displayed, on the DIVA Node (G6.Node) window, click the Configuration button.
    • Result: The Configuration window is displayed.
  2. In the Hostname field, enter the name of the host.
    • Note: This is the name of the host that is running DIVA Server.
  3. Leave the port number and check in period as is.
    • Note: There is no need to change port number and check in period.
  4. Click OK.
    • Result: The new configuration is applied and the node will reconnect to the server, and the DIVA Node window is displayed.

What's Next?

On the DIVA Node window check the following:

  • The new node's host name and IP address should be displayed next to the Configuration button.
  • The server host name and server status should be displayed at the bottom-left of the window.

Return to DIVA Fusion and using the browser's Refresh button, refresh the node's Properties page. This will populate the following fields:

  • Hostname
  • IP Address
  • Processor
  • Physical Memory
  • Enabled
  • Online
  • Last Seen

Note: The Volumes tab displays all the physical drives on the node; not the DIVA drives.
